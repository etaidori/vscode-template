# My VS Code Extension

This is a template for creating VS Code extensions.

## Getting Started

1. Edit `package.json` to customize your extension metadata
2. Implement your extension logic in `src/extension.ts`
3. Run `npm install` to install dependencies
4. Run `npm run compile` to build
5. Press F5 to launch the extension in a new VS Code window

## Development

- `npm run watch` - Watch for changes and compile automatically
- `npm run package` - Build and package the extension as VSIX
- `npm run lint` - Run linter
- `npm run test` - Run tests

## Publishing

To publish to the VS Code Marketplace, use:
```
vsce publish
```

Ensure you have a publisher account set up first.
